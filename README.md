# Touchway Kiosk Releases

This repository is public and is used only to distribute release artifacts for Touchway kiosk applications.

## Purpose

- Hosts signed/built release binaries and update metadata used by kiosk devices.
- Serves as the public update source for unattended production deployments.

## Important

- Do not add source code, credentials, private keys, or internal documentation.
- Do not open pull requests with sensitive operational details.
- Artifacts are published automatically by CI from the private source repository.

## For Operators

If you need to bootstrap a kiosk device, use the installation scripts from the private source repository and point devices to the official release channel.

## Security Notice

If you find a security issue, report it through internal security channels. Do not disclose vulnerabilities in this public repository.
